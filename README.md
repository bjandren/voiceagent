# AI-Driven Customer Support Agent for Real Estate

En SaaS-baserad AI-kundsupportagent som hanterar telefonsamtal från hyresgäster för fastighetsbolag.

## Projektöversikt

Detta system utnyttjar en förbyggd kunskapsbas och avancerad röstsyntesteknik för att besvara rutinförfrågningar, samtidigt som det intelligent eskalerar ärenden baserat på deras karaktär. Det är designat för att stödja flera samtidiga samtal och integreras sömlöst med befintliga fastighetsförvaltningssystem.

### Kärnfunktioner

- Automatiserad hantering av hyresgästförfrågningar via naturligt språkgränssnitt
- Intelligent ärendeeskalering för administrativa ärenden, fastighetsproblem eller akuta nödsituationer
- Robust, anpassningsbar lösning för fastighetsbolag som integreras med deras befintliga arbetsflöden

## Teknisk stack

- **Frontend:** React på Vercel med Shadcn komponentbibliotek
- **Backend:** Python (FastAPI/Flask) med asynkron bearbetning
- **Databas:** Supabase (PostgreSQL) med Drizzle ORM
- **Röstteknologi:** Elevenlabs för röstsyntesering
- **Telefoni:** Twilio för samtalshantering

## Projektstruktur

- `/src` - Frontend-kod (React)
- `/app` - Backend-kod (Python)
- `/services` - Mikrotjänster (Node.js)
- `/config` - Konfigurationsfiler
- `/tests` - Testfiler
