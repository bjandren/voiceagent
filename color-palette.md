# Color Palette

## Overview

Define a modern, consistent color palette that aligns with the brand identity and UI aesthetics of the AI customer support agent platform.

## Primary Colors

- **Primary Blue:** #1E90FF
  - Used for primary buttons, headers, and active elements.
- **Primary Dark Blue:** #0B3D91
  - Used for navigation bars and footer backgrounds.

## Secondary Colors

- **Secondary Light Gray:** #F5F5F5
  - Used for backgrounds and subtle accents.
- **Secondary Gray:** #B0B0B0
  - For secondary buttons, borders, and inactive elements.

## Accent Colors

- **Accent Green:** #32CD32
  - For success messages, confirmations, and status indicators.
- **Accent Red:** #FF4500
  - For error messages, urgent notifications, and alerts.
- **Accent Yellow:** #FFD700
  - For warning messages and attention indicators.

## Supporting Colors

- **Neutral White:** #FFFFFF
  - Backgrounds, cards, and modals.
- **Neutral Black:** #000000
  - Text, icons, and primary content.

## Usage Guidelines

- Ensure sufficient contrast for accessibility.
- Primary colors dominate calls to action and interactive elements.
- Secondary and accent colors are used sparingly to guide attention without overwhelming the user.
- Maintain consistency across the dashboard and telephony UI components.
